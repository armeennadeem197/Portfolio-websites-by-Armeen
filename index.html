Latest Project

Flowers website

This website showcases a beautifully designed web application dedicated to the world of flowers. It features an extensive collection of floral varieties, each with detailed descriptions, care tips, and stunning images. Users can easily browse and filter flowers by type, color, and seasonal availability. The application also includes an interactive section where users can learn about the symbolism of different flowers and create personalized arrangements for various occasions. The responsive design ensures an optimal viewing experience on both desktop and mobile devices, making it accessible to flower enthusiasts everywhere. Through this project, I demonstrated my skills in web development, UI/UX design, and attention to detail.

Working Amozon website

This website is a clone of the popular e- platform, Amazon, featuring a user-friendly interface and robust functionality. Users can browse an extensive catalog of products, filter items by category, price range, and ratings, and add them to their shopping cart. The application includes a secure authentication system, allowing users to create accounts and manage their profiles. Additionally, it incorporates a payment gateway for seamless transactions and a review system for customer feedback. The responsive design ensures an optimal experience across all devices, enhancing usability and engagement. This project showcases my skills in full-stack development, database management, and responsive design principles.
